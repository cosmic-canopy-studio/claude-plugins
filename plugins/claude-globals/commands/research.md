# Comprehensive Research Request

Research the following topic thoroughly: $ARGUMENTS

## Research Requirements

1. **Breadth**: Cover multiple perspectives and source types
2. **Depth**: Go beyond surface-level information
3. **Recency**: Prioritize recent information where relevant
4. **Credibility**: Use high-quality sources
5. **Synthesis**: Combine findings into coherent insights

## Process

Think about this research question deeply, then:

1. Break down the topic into 3-5 research angles
2. Use specialized subagents in parallel:
   - topic-researcher for comprehensive exploration
   - academic-researcher for scholarly perspectives
   - news-researcher for current developments
   - data-analyst for statistics and trends
3. Have citation-manager verify all sources
4. Use synthesis-expert to create final report

Create research plan in TodoWrite before starting.
