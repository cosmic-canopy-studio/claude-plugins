# Academic Research Request

## Academic Research Requirements

1. **Primary Focus**: Peer-reviewed academic sources
2. **Secondary Support**: Authoritative institutional sources
3. **Recency**: Prioritize last 5 years, include seminal works
4. **Methodology**: Document research methods and limitations
5. **Synthesis**: Academic literature review format

## Process

Think carefully about this academic research question, then:

1. Primary: Use academic-researcher subagent for peer-reviewed sources
2. Support: Use topic-researcher for background context
3. Support: Use news-researcher for recent developments/clinical trials
4. Citation-manager for academic citation formatting
5. Synthesis-expert for literature review creation
